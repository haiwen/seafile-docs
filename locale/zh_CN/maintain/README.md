# 管理员

### 进入管理界面

作为系统管理员，你可以通过点击网页左上侧的`tools`按钮（在个人头像旁)进入管理界面：

![管理工具](../images/tools-button.png)

点击完`tools`按钮，你便进入管理界面：

![系统管理界面](../images/system-admin.png)
